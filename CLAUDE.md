# CLAUDE.md — הקשר פרויקט ל-Claude Code

קובץ זה נטען אוטומטית על ידי Claude Code בתחילת כל סשן. הוא מתאר את הפרויקט,
את הידע התחומי, את האילוצים ואת מוסכמות העבודה. עדכן אותו ככל שהפרויקט מתפתח.

## מה הפרויקט

ערכת כלים לניתוח השקעות ורכישת דירות בישראל, נכון ל-2026. שני רכיבים עיקריים:

1. **מנוע חישוב** (`src/realestate/`) — חבילת פייתון המחשבת מס רכישה, מימון
   ומשכנתא, תשואות, לוח סילוקין ותחזית מכירה כולל מס שבח.
2. **מנוע נתוני שוק** (`src/realestate/nadlan.py` + `scripts/market.py`) —
   מושך נתוני עסקאות שנמכרו מאתר הנדל"ן הממשלתי לפי אזור.
3. **מחשבון אינטראקטיבי** (`web/index.html`) — כלי דפדפן עצמאי, כולל הדבקה חכמה
   של מודעות ודירוג עסקאות.

## מבנה

```
src/realestate/    חבילת הליבה (config, tax, finance, parser, nadlan)
scripts/market.py  CLI למשיכת נתוני שוק
web/index.html     מחשבון דפדפן עצמאי (דו-לשוני עברית/English)
tests/             בדיקות pytest
data/              פלט נתונים (לא נכנס ל-git)
```

## הרצה

```bash
pip install -r requirements.txt
python -m pytest                      # בדיקות
python scripts/market.py "רמת גן"     # משיכת נתוני שוק
```

## ידע תחומי — קבועים רגולטוריים 2026

המקור היחיד לאמת לכל הקבועים הוא `src/realestate/config.py`. אל תשכפל מספרים
בקוד אחר — תמיד ייבא מ-config.

- **מס רכישה — דירה יחידה** (קפוא עד 15.1.2028): 0% עד 1,978,745 ₪; 3.5% עד
  2,347,040; 5% עד 6,055,070; 8% עד 20,183,565; 10% מעבר.
- **מס רכישה — דירה נוספת/השקעה**: 8% עד 6,055,070; 10% מעבר (מהשקל הראשון).
- **תקרות מימון (בנק ישראל)**: דירה יחידה 75%, משפר דיור 70%, השקעה 50%.
- **מס שבח**: 25% על השבח ה**ריאלי** (אחרי ניכוי אינפלציה), 30% עם מס יסף
  לבעלי הכנסה גבוהה. דירת מגורים יחידה פטורה עד תקרת 5,008,000 ₪ (קפוא עד 2028).
- **מע"מ**: 18% (מ-2025), חל על שירותי תיווך ועו"ד.
- **יחס החזר (DTI)**: תקרה רגולטורית 50% (40% מצרפי מ-1.7.2026); בנקים מאשרים
  בפועל 30%-35% מההכנסה הפנויה. ברירת המחדל בכלי (33%) היא אומדן.
- **ריבית משתנה**: חלק המסלולים בריבית משתנה (פריים) לא יעלה על 66.66% מההלוואה.

כשמתעדכנת רגולציה: עדכן את `config.py` בלבד, הרץ בדיקות, עדכן ערכי ציפייה בטסטים.

## אילוצים — חשוב מאוד

- **אסור לבנות scraper ליד2 או מדלן.** תנאי השימוש שלהם אוסרים איסוף נתונים
  אוטומטי במפורש (רובוטים/spiders/scrapers), והאתרים מוגנים בזיהוי בוטים.
  אין לעקוף הגנות בוטים או CAPTCHA. אם נדרש מידע על מודעות חיות — הדרך החוקית
  היא פיד מורשה דרך מעמד תיווך, או הדבקה ידנית של מודעה בודדת שהמשתמש גולש בה.
- **מקור הנתונים הלגיטימי לעסקאות שנמכרו** הוא אתר הנדל"ן הממשלתי
  (nadlan.gov.il) — מאגר פתוח לעיון הציבור. זה מה ש-`nadlan.py` משתמש בו.
- הממשק של nadlan.gov.il אינו רשמי ועשוי להשתנות; שמור על קוד הרשת מבודד
  (`make_session`/`resolve_area`/`fetch_deals`) כדי שיהיה קל לתקן ולבדוק עם mock.

## דיסקליימר

הכלי להמחשה בלבד ואינו ייעוץ פיננסי, מיסויי או משכנתאי. חישובי מס שבח מפושטים.

## מוסכמות קוד

- פייתון 3.10+, type hints, dataclasses לקלט/פלט.
- כל לוגיקת חישוב נטו (ללא רשת/קבצים) חייבת להיות מכוסה בבדיקות pytest.
- בקוד הפייתון: שמות פונקציות/משתנים באנגלית, הערות בעברית.
- **כל הפלט מול המשתמש דו-לשוני (עברית/English):** מחשבון ה-HTML (כפתור החלפת
  שפה + RTL/LTR), חוברת ה-Excel (תוויות כפולות), ומנוע הנתונים (Excel + מסוף).
- אין תלויות כבדות מיותרות — כרגע רק requests ו-openpyxl.
